== Description ==
This plug-in changes the tab title and favicon when your visitors skip to another tab.
Thanks, Don't GO JS : https://github.com/tiaanduplessis/dont-go

== Installation ==
Please follow these steps to install the plugin.

1. Upload `wp-dontgo` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `WP Don't GO` menu in WordPress
